# radio.lucafchala.com

Portal comunitário dedicado ao radioamadorismo, APRS, RF, modos digitais, experimentação e infraestrutura de comunicação resiliente.

## Objetivo

O projeto busca criar um espaço simples, rápido e permanente para hospedagem de páginas relacionadas à comunidade radioamadora.

Toda a infraestrutura é baseada em arquivos estáticos (`HTML/CSS/JS`), priorizando:

- simplicidade
- confiabilidade
- baixa manutenção
- velocidade
- independência
- compatibilidade de longo prazo

Inspirado na internet clássica e no espírito experimental do radioamadorismo.

---

## Infraestrutura

- Cloudflare Pages
- Cloudflare CDN
- Proteção DDoS
- HTTPS automático
- Deploy automatizado via GitHub
- HTML/CSS/JS puro
- Sem frameworks
- Sem backend
- Sem banco de dados

---

## Deploy

O projeto utiliza:

```txt
GitHub → Cloudflare Pages
```

Cada push no repositório gera automaticamente um novo deploy.
